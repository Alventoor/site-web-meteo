Bienvenue

Ce site a été conçu pour vous permettre de visualiser les données météorologiques de différents endroits de France. Pour cela rien de plus simple, il vous suffit de sélectionner depuis le menu l'un des deux modes de visualisation.

Visualisation en direct

Cette page vous permet de visualiser les toutes dernières données météorologiques d'un lieu. Les données affichées sont reçues en temps réel sans qu'aucun rafraîchissement de la page ne soit nécessaire.

Évolution des températures

Cette page vous permet de visualiser les données météorologiques sous forme de graphique sur une période choisie.
